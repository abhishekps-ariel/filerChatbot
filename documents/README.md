# Petition Documents Folder

Store your petition PDF documents here. These will be ingested by the admin for the Q&A bot.

## Usage

1. Place all petition-related PDF files in this folder
2. Run the ingestion script to process them:
   ```powershell
   cd ..
   python ingest_documents.py
   ```

## Naming Convention

For better organization, name your files clearly:
- `petition_guidelines.pdf`
- `petition_categories.pdf`
- `petition_faq.pdf`
- `petition_process.pdf`
